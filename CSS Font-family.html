Thuộc tính font-family

HTML

CSS

Để lựa chọn typeface cho chữ, văn bản bằng CSS, chúng ta sử dụng thuộc tính font-family.

Ví dụ:


body { font-family: Arial; }
tt { font-family: Courier, monospace; }
p { font-family: "Trebuchet MS", Verdana, sans-serif; }

Ngoài 5 loại generic font family: serif, sans-serif, monospace, cursive, fantasy - mọi tên font khác cần được viết hoa. Ví dụ như “Arial” thay vì “arial”.

Những tên font có chứa dấu cách như Trebuchet MS trong ví dụ trên, chúng ta cần thêm dấu ngoặc kép bên ngoài.

Nếu có nhiều hơn một tên font, giữa các tên font được ngăn cách bởi dấu phẩy.

Bạn có thể xem thêm về 5 loại generic font family tại trang web W3School.

Vì sao chúng ta lại sử dụng nhiều hơn một font ?

Các trình duyệt không thể hiển thị các kiểu font mà người dùng chưa cài đặt trong máy tính. Khi kiểu font mà chúng ta sử dụng không được cài đặt trên thiết bị của người dùng, kiểu font mặc định của trình duyệt sẽ được hiển thị thay thế.

Chúng ta nên lựa chọn font như thế nào?

Hãy bắt đầu với lựa chọn của bạn, sau đó thêm vào những kiểu tương tự để thay thế, và kết thúc với một generic font family.

Ví dụ: 


body { font-family: Verdana, Arial, Helvetica, sans-serif; }

Ngoài ra, trình duyệt Google hỗ trợ người dùng với hơn 1000 font miễn phí, hiển thị trực tiếp trên trình duyệt mà không cần người xem cài đặt trong máy tính.

Xem thêm về cách cài đặt CSS Google Font tại trang web W3School.
